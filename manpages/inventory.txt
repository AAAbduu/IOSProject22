NAME: 
        inventory  -  lists existing item
SYNOPSIS:
        inventory
        inventory [item] 

DESCRIPTION: 
        inventory : lists the existing objects in your inventory.
        inventory + [object] : displays the content of the chosen file, if it exists, like the description of the usefulness and objective of this object,
        otherwise if the object does not exist, informs that the object does not exist in your inventory.

SYSTEM CALLS:
        open, read, write, close, readdir

C LIBRARY FUNCTIONS:
        opendir, closedir

RETURN VALUE:
        'This is what you are carrying:   '
        or
        'There is no such object in your inventory'
        or
        'description
        
EXCEPTION:
        inventory kid
        returns-> I should not torture the kid....
