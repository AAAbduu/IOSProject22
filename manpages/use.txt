NAME: 
        Use  -  use an item from the inventory 
SYNOPSIS:
        use [object]

DESCRIPTION: 
        Use an existing object in the inventory, in a place or a character. 
        If the object does not exist in the inventory a message indicating it will be displayed, 
        otherwise the object is used and removed from inventory.
        
SYSTEM CALLS:
        open, unlink

RETURN VALUE:
        'I don't have that object'
        or
        Operation will be executed and a message will be displayed in the function of the case
