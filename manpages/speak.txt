NAME: 
        speak       talk to a character

SYNOPSIS:
        speak
        speak [character]

DESCRIPTION: 
        When we write speak without any argument, it shows us the characters with whom we 
        can chat in a given place.
        But when we write speak followed by a character it shows us the words exchanged 
        with this character.

SYSTEM CALLS:
        open, read, write, close, readdir, lseek

C LIBRARY FUNCTIONS:
        opendir, closedir

RETURN VALUE:
        Scenario swapped
